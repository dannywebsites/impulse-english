## CRITICAL: Template Cleanliness Failures

The integration checker found the template is **not clean for hand-off**. A new client who clones and runs `npm run onboard` will deploy a site with Impulse English Academy branding in dozens of locations.

### Critical (4 issues — would cause data leakage or broken functionality)

| ID | File(s) | Issue |
|----|---------|-------|
| CRIT-01 | `BaseLayout.astro` | Hardcoded: SITE_NAME, BASE_URL, GA4 ID (G-WN5973VY1M), Google Ads ID (AW-11461982741), GTM container (GTM-TDC7CQDD), @ImpulseEnglish Twitter handle. **Tracking IDs would feed new client's lead data to Impulse's analytics.** |
| CRIT-02 | `SEOHead.tsx` | Hardcoded: SITE_NAME, BASE_URL, author. Duplicate constants not reached by CLI. |
| CRIT-03 | `auto-publish.js` | Hardcoded BRAND block + 30 S3 image URLs. Completely disconnected from brand-config.ts. **SEO pipeline would generate Impulse-branded articles for new client.** |
| CRIT-04 | `site.webmanifest` | Impulse name + broken `/images/optimized/` icon paths (directory deleted Phase 3). **PWA icons 404.** |
