If the build fails, the generators have syntax errors in their template strings. Common issues to check and fix in `onboard-client.js`:
- Missing `as const` at end of objects
- Missing `export type BrandConfig = typeof BRAND_CONFIG;` at end of brand-config
- Missing or malformed helper functions at end of napData
- Unescaped quotes or backticks in template literals
- Missing commas between object properties
- Missing newlines between sections
- Array formatting issues (trailing commas, missing brackets)

Fix any issues found in `onboard-client.js` and re-run until the build passes.
